# Bun POC

- Fast all-in-one JavaScript runtime

## Install

- [Bun](https://bun.sh/)
- `curl https://bun.sh/install | bash`
- .bash_profile
```
export BUN_INSTALL="/Users/chesterheng/.bun"
export PATH="$BUN_INSTALL/bin:$PATH"
```

## Features

- Bundle, transpile, install and run JavaScript & TypeScript projects
